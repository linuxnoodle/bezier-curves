# bezier-curves
An SDL2 program used to generate bezier curves. (Photoshop pen tool/SVG graphics)
# Requirements:
- SDL2, c++17 (could probably be used on earlier versions of c++)
# Building;
```
git clone https://github.com/linuxnoodle/bezier-curves
cd bezier-curves
make clean all
```
